hTab - Help & Keyboard Shortcuts

Navigation

↑ / ↓ : Move selection line

PgUp / PgDn : Scroll page up/down

Home / End : Jump to first/last process

Space : Tag / Untag selected process

Enter : Focus / Switch to selected tab

Actions

F1 / 1 / h : Show Help

F2 / 2 / s : Open Setup (Themes)

F3 / 3 / / : Search tab command

F4 / 4 / \ : Filter process list

F5 / 5 / t : Toggle Tree View (by Domain)

F6 / 6 / o : Sort by column

F9 / 9 / k : Kill selection (Close tab)

F10 / 0 / q : Quit (Close new tab)

hTab v1.0.0 - A cross-browser extension developed using Manifest V3.

Processes labeled with real domain URLs are actual active browser tabs! Killing them closes the tab instantly. Background tasks are simulated browser sub-systems.
